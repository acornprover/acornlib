# Measure Theory

Goal: build the measurable and integrable foundations needed by probability, analysis, and geometry.

- [ ] Trace sigma-algebra follow-ups: trace of a generated sigma-algebra equals the generated trace; trace commutes with `borel_sigma_algebra` on a subspace; measurability of the inclusion-restriction of a measurable function into the trace
- [ ] Borel sigma-algebra generated by a topological subbasis: show it equals `borel_sigma_algebra` when the subbasis generates the topology
- [ ] Develop outer measures and Caratheodory extension
- [ ] Add measures, finite measures, and signed measures
- [ ] Support null sets and almost-everywhere reasoning
- [ ] Introduce simple functions and nonnegative integration
- [ ] Add Bochner-style integration in normed spaces
- [ ] Develop product measures, Fubini, and Tonelli
- [ ] Add regularity results and canonical measures such as Lebesgue measure
